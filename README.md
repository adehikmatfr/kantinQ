## KantinQ

sebuah design UI yang di bangun menggunakan react-native.

## Required

- Nodejs
- android studio or device
- yarn

## How To Use

```
$git clone https://github.com/adehikmatfr/kantinQ.git;
$yarn install
$yarn start
$yarn run android
```

fork untuk mendapatkan updatean aplikasi . <br/>
follow agar dapat pemberitahuan aplikasi open source saya! terimakasih...
